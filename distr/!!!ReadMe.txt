ОАИСКГН. ПО УПК
Настройка ИТО (измеритель тяжения оптический), получение измерений, расчет значений температуры, тяжения и гололеда, усреднение значений и отправка на сервер ОСМ.

Для установки службы:
1. Убедиться, что дистрибутив находится в папке C:\OAISKGN_UPK\distr
2. В файле MakeWinService.bat проверить IP-адрес в строке nssm set OAISKGN_UPK AppParameters "192.168.1.94 7681". При необходимости изменить на действительный адрес компьютера УПК
3. Запустить cmd от имени Администратора
4. Ввести команду "cd C:\OAISKGN_UPK\distr"
5. Ввести команду "MakeWinService.bat"
6. Проверить наличие службы "ОАИСКГН. ПО УПК" в диспетчере локальных служб ОС Windows

Для удаления службы:
1. Остановить службу "ОАИСКГН. ПО УПК" в диспетчере локальных служб ОС Windows
3. Запустить cmd от имени Администратора
4. Ввести команду "cd C:\OAISKGN_UPK\distr"
5. Ввести команду "nssm remove OAISKGN_UPK"
